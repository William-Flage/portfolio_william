This Portfolio will contain information about my education, skills, and achievements in school and the field of programming.

Web Design: Unit 2

Over my second unit of Web Design, I mainly learned about the simple tags that you can use in html to format your website, such as lists or tables. Here are some things I found particularly interesting.

  • There are several kinds of tags that format even basic layouts
  • Lists can be made using numbered or bulleted formats
  • Links can be created for different websites, web pages in your website, or different locations on the same page
  • Data tables are very useful for portraying information and can be created with a large amount of customization

Web Design: Unit 3

During my third unit of Web Design, I learned much more about CSS, including how to style different elements, such as paragraphs, lists, tables, and more. I also learned about organizing and shaping a website using margins, padding, and text-align to create a professional looking website. Here are some things I found particularly interesting.

  • You can use the colors of a website to help convey it's message and theme.
  • You can use CSS to change the color of different parts of the same element, such as the background color, text color, or margin color.
  • CSS can be used to completely change the look of an element, such as when I changed my Nav from a simple unordered list, into a set of buttons above the main body.
  • You can change the style of individual elements of the website, such as the style of the margin, the style of the text, or even the style of embedded links.

Web Design: Unit 4

In the fourth unit of Web Design, I learned about graphics and how to apply them. This includes uploading and cropping images, linking to a full image size, and also how to apply a Favicon to a website. Additionally, I learned about copyright and how to properly find free use images. Below are a few things that I thought were interesting during Unit 4.

  • Any image, idea, or creation that is unique is copyrighted by the creator, at the moment of creation.
  • You can crop images to multiple different sizes to better fit your website, and can place links inside of images to make them redirect you to other webpages.
  • You can use a 16x16px image to create a favicon, which is the icon beside the tab for each webpage.

Web Design: Unit 5

Over my fifth unit of Web Design, I learned about javascript. This included how to code, where to place the script element, and possible applications of javascript. Below are some things I found interesting about javascript.

  • Javascript can be used to create popup messages, and these messages can be used for conveying errors, asking for confirmation, or for conveying information
  • Javascript can be used to create a clock that updates in real time, as well as more updating information
  • You can hide elements using javascript, which could be useful for menus, lists of information, or filtering already existing information.
