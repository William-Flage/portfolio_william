Evaluation of Web Graphics

www.canva.com

The purpose of the Graphics in this website is to convey the types of designs that you can create, and the possible purpose those designs might have. These graphics meet the following Guidelines for effective use of web graphics:

  1. The graphics fit the purpose, style, and organization of the page by using a similar color palette, assisting in navigation, and displaying the site's purpose.
  2. The website uses some complex image files, adding to the load time of the website.
  3. The graphics focus the viewer's attention to links that lead to important functions of the website.
  4. The website avoids repetitive imagery, and avoids bright or suprising graphics that can draw the viewer's attention away from it's intended focus.
  5. Each graphic is accomonied by text that conveys the same meaning, so it does not rely on the graphics to convey information.
  6. The graphics do not contain alt text that is descriptive.
  7. Text color contrasts nicely with background color to allow the viewer to read the site easily.
This website meets 5 of the 7 rules, and fails at rule 2 and 6.
